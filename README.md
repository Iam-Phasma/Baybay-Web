# Baybay — Baybayin Learning App Landing Page

A heritage-inspired landing page for **Baybay**, a free Android application dedicated to learning and practicing Baybayin—the ancient Filipino script—through interactive lessons, quizzes, transliteration tools, and memory games.

## About

Baybay is a cultural learning project that makes Filipino script exploration practical, approachable, and free. The app combines guided lessons with engaging gameplay to help learners of all levels master Baybayin.

**Current Version:** 3.8.3  
**Released:** December 2023

## Features

- **Guided Lessons** — Learn Baybayin characters and their usage systematically
- **Quizzes** — Test your knowledge with interactive assessments
- **Transliteration Tools** — Convert between Baybayin and modern Filipino scripts
- **Memory Games** — Reinforce learning through engaging gameplay
- **Dark/Light Mode** — Comfortable viewing in any lighting condition
- **Responsive Design** — Optimized for all screen sizes

## Tech Stack

- **HTML5** — Semantic structure and accessibility
- **CSS3** — Design system with warm heritage color palette
- **JavaScript** — Theme persistence, carousel interactions, and animations
- **Bootstrap 5.3** — Responsive grid and utilities
- **Google Fonts** — Montserrat, Lora, and Noto Sans Tagalog typography

## Setup & Deployment

This is a static website—no build process required.

### Local Development

1. Clone or download this repository
2. Open `index.html` in your browser
3. Edit files as needed; changes appear immediately

### Deployment

- **GitHub Pages:** Push to a repository and enable Pages in settings
- **Any static host:** Upload all files to your server
- No backend or server-side processing needed

## Theme System

The project uses a sophisticated design token system reflecting Baybayin heritage aesthetics:

- **Light Mode:** Warm cream background (#FAF4E6) with deep brown text and gold accents
- **Dark Mode:** Deep charcoal with pale cream text and warm gold highlights
- User preference persists across sessions via localStorage

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Responsive on desktop, tablet, and mobile
- No external dependencies except Bootstrap and Google Fonts
